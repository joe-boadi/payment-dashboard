# Customer Payments Web Application

- Assignment for PSC-GH – Submission: 14 March 2026


## Features
- Real-time table from an `API`
- Date range filter + search + sorting
- Modal with full details (ModeOfPayments, invoices, Payee)
- Fully responsive + accessible
- CORS handled via Vite proxy

## Setup
1. `npm install`
2. Copy `.env.example` → `.env`
3. `npm run dev`

## Tech
React 18 + Vite - Vanilla CSS - JavaScript - Lucide icons - PropTypes - JSDoc
